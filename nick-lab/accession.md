# Editor
Der Editor wird aufgerufen, 
* indem man in der Listen Ansicht die ID einer Akzession anklickt
* indem man im Verwaltungsmenü auf Bearbeiten klickt und anschließend die ID der gewünschten Akzession eingibt.
